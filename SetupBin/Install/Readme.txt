A. Kelengkapan media
Karena ukuran besar beberapa file sumber instalasi platfor tdk diinclude. Paket setup ini perlu dilengkapi dengan: 
1. source\sxs yang diambil dari Win8 CD source\sxs
2. Folded DB diisi dengan backup DB terbaru dengan nama:
    - ASPState.bak
	- EWP_V2.bak
3. Folder WebApp diisi dengan files-file web aplikasi yang telah dipublish. (Publish lah CoreCMS ke folder ini 

B. Petunjuk instalasi
1. Untuk Windows 7 dan Windows Server 2008:
   1. Jalankan PreSetup.exe dg "Run As Administrator"
   2. Jalankan Setup.exe

2. Untuk Windows 8, 8.1 dan Windows Server 2012
   1. Buka command promt dengan "Run As Administrator"
   2. Pindah drive ke drive instalasi (mis: E:\
   3. pindah direktori ke folder Win8 (mis: cd E:\SetupBin\Win8
   4. Jalankan PreSetupWin8.Bat
   5. Selanjutnya ikuti langkah seperti untuk Windows 7 diatas
